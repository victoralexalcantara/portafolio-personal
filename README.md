# Proyecto de Portafolio con IA

Este es un proyecto de Next.js iniciado en Firebase Studio que utiliza la IA de Google para mejorar el contenido de un portafolio.

## Empezando

Estas instrucciones te permitirán obtener una copia del proyecto en funcionamiento en tu máquina local para fines de desarrollo y prueba.

### Prerrequisitos

Necesitarás tener Node.js y npm (o un gestor de paquetes similar) instalados en tu sistema.

* [Node.js](https://nodejs.org/)
* [npm](https://www.npmjs.com/get-npm)

### Instalación

1.  Clona el repositorio:
    ```sh
    git clone https://github.com/tu_usuario/tu_repositorio.git
    ```
2.  Instala las dependencias del proyecto:
    ```sh
    npm install
    ```
3.  Inicia el servidor de desarrollo:
    ```sh
    npm run dev
    ```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador para ver la aplicación.

## Características

*   **Mejora de contenido con IA:** Utiliza Genkit y la IA de Google para analizar y sugerir mejoras en el texto del portafolio.
*   **Validación de formularios:** Usa `zod` para validar las entradas del usuario en el lado del servidor.
*   **Componentes reutilizables:** Construido con un sistema de componentes de React para una fácil personalización y mantenimiento.

## Despliegue

Este proyecto está configurado para ser desplegado fácilmente en Firebase Hosting.
