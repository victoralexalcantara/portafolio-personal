# **App Name**: Victor's Vision

## Core Features:

- Dynamic Portfolio Display: Display portfolio projects with titles, descriptions, and technologies.
- Responsive Layout: Ensure the portfolio is fully responsive across all devices.
- Interactive Skill Showcase: Create an interactive display of key skills and technologies with a brief explanation of proficiency and experience in each.
- AI-Powered Content Improvement: Integrate an AI tool to refine the portfolio content and suggest improvements. For instance, suggesting word improvements based on a professional portfolio style guide.
- Contact Form: Implement a functional contact form for inquiries.
- Firebase Integration: Use Firebase Hosting for deployment.
- Auth0 integration: Authenticate access via Auth0.

## Style Guidelines:

- Primary color: Deep blue (#2E5266) to evoke professionalism and trust. 
- Background color: Light gray (#E0E0E0) for a clean and modern look. 
- Accent color: Subtle orange (#D98E52) to highlight important sections and calls to action. 
- Headline font: 'Belleza', sans-serif, for a design-forward presentation
- Body font: 'Alegreya', serif, to establish legibility when describing work history
- Use clean and professional icons for social media links and sections.
- Design a responsive layout that adapts seamlessly to different screen sizes, ensuring optimal viewing experience across all devices.
